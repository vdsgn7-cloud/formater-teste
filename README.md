# Gerador de Roteiro de Estudo — Colégio Cristo Rei

## Deploy no Vercel

1. Suba esta pasta para um repositório GitHub novo
2. Importe no [vercel.com](https://vercel.com) → Add New Project
3. **Framework Preset**: Next.js  
4. **Root Directory**: deixe vazio (raiz do repo)
5. Clique Deploy — não precisa de variável de ambiente

## Desenvolvimento local

```bash
npm install
npm run dev
```
Acesse: http://localhost:3000
